## 📖 How to Download a Video

Once the app is open, you will see a prompt. Follow these instructions to pull a video from the web:

1. Copy the URL of the video from your web browser.
2. Paste the link into the application window and press Enter.
3. The app checks the link and shows the available format options.
4. Pick the number that matches your preferred quality.
5. Wait for the progress bar to finish.
6. The app saves the file in the same folder where the program resides.

## 📂 Troubleshooting

If the program closes unexpectedly, follow these steps:

*   Restart your computer.
*   Verify that you have an active internet connection.
*   Ensure that no other program is using the same file.
*   Check that your antivirus software is not blocking the action. 

The software includes basic error logging. If you see text in red color, read it carefully as it often explains why a download did not start. Most errors happen due to network timeouts or invalid video links provided by the user. Ensure your link starts with https:// to avoid issues. 

For better performance, use a wired network connection when downloading long videos or high-resolution files. This reduces the chance of interrupted transfers. Keep your Windows system updated to ensure full compatibility with the latest tools and features.
